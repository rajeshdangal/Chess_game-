About the project

We made this page to present our chess game clearly for teachers and reviewers.

♟

Purpose

We built this chess game in Python with Pygame to show gameplay, design, and structured code.

♜

Current state

The project already includes board drawing, piece setup, movement logic, turn handling, and a save system.

♚

Goal

Our aim was to create a playable chess game with local multiplayer support and a clean presentation page.

Features

These are the main features included in our project.

♞

Menu system

A start menu lets the user begin the game and move through the available options.

♕

Chess logic

The game handles board rules, turn switching, and valid movement for the pieces.

⬇

Save and load

The game state can be stored and continued later from a saved session.

🌐

Local multiplayer

We designed the project to support multiplayer on the same network using Python sockets.

System architecture

This shows the object-oriented structure of the application.

UML class diagram

🧠

Game core

The Game and Scene classes control the flow of the application, menu transitions, and active game state.

🧩

Board and pieces

The Board, Tile, and Piece classes manage placement, movement rules, and rendering.

🌐

Networking and save

The SaveSystem stores game state, and the NetworkManager supports LAN multiplayer communication.
